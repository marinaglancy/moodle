Video.JS 5.12.6

This is a sample file to demonstrate double controls in videos on iPhone. This file does not include any custom CSS.

Do the following (on iPhone):

  • Play video clicking on play button, it will open in fullscreen
  • Exit full screen, video will continue playing inline
  • Click on the video
  • Now you can see video.js controls but you also see native controls
  • If you resume video - it will play inline without opening full screen

This means that videos CAN play inline on iPhone, the only problem is now to make it not enter fullscreen when play button is clicked for the first time

See here for an example with custom CSS that removes native video controls on iPhone.
